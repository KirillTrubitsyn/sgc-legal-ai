# Руководство пользователя SGC Legal AI

## Что это за приложение?

**SGC Legal AI** — это интеллектуальный помощник для анализа юридических вопросов в области российского права. Приложение использует несколько передовых языковых моделей (искусственный интеллект) для подготовки правовых заключений и автоматически проверяет ссылки на судебную практику через официальные реестры.

### Для кого предназначено?

Приложение разработано для сотрудников юридической службы Сибирской генерирующей компании. Доступ осуществляется по персональному инвайт-коду.

---

## Начало работы

### Вход в систему

1. Откройте приложение в браузере
2. На главной странице введите ваш **инвайт-код** в поле ввода
3. Нажмите кнопку **«Войти»**
4. После успешной авторизации вы будете перенаправлены в чат

> **Примечание:** Инвайт-код выдается администратором системы. Если у вас нет кода — обратитесь к администратору.

---

## Основной интерфейс чата

После входа вы попадаете на главный экран — **Чат**. Здесь происходит вся работа с системой.

### Элементы интерфейса

| Элемент | Расположение | Назначение |
|---------|--------------|------------|
| Логотип | Верхний левый угол | Переход на главную страницу чата |
| Кнопка «Сохранённые» | Верхняя панель | Переход к списку сохранённых ответов |
| Имя пользователя | Верхняя панель справа | Отображение текущего пользователя |
| Кнопка «Выйти» | Верхний правый угол | Выход из системы |
| Переключатель режимов | Под заголовком | Выбор между «Консилиум» и «Запрос» |
| Кнопка «Новый чат» | Панель управления | Очистка истории и начало нового диалога |
| Поле ввода вопроса | Нижняя часть экрана | Ввод вашего юридического вопроса |
| Кнопка загрузки файла | Рядом с полем ввода | Прикрепление документов к запросу |

## Два режима работы

Система предлагает два режима анализа — выберите подходящий в зависимости от сложности вопроса.

### Режим «Запрос»

Быстрый анализ вашего вопроса одной моделью искусственного интеллекта.

**Когда использовать:**
- Для простых и типовых вопросов
- Когда нужен быстрый ответ
- Для уточняющих вопросов в рамках диалога

**Настройки режима:**

| Опция | Описание |
|-------|----------|
| **Быстрый** (иконка молнии) | Оперативный ответ за несколько секунд |
| **Думающий** (иконка мозга) | Глубокий анализ с расширенным размышлением (занимает больше времени) |
| **Поиск судебной практики** (переключатель) | Включает автоматический поиск и верификацию судебных дел |

**Как использовать:**
1. Выберите режим **«Запрос»** в переключателе
2. Выберите тип: «Быстрый» или «Думающий»
3. При необходимости включите поиск судебной практики
4. Введите ваш вопрос в поле ввода
5. Нажмите Enter или кнопку отправки

---

### Режим «Консилиум»

Комплексный анализ вопроса несколькими моделями искусственного интеллекта с последующей выработкой консенсуса.

**Когда использовать:**
- Для сложных правовых вопросов
- Когда важна всесторонняя проработка
- Для ответственных юридических заключений
- Когда нужны ссылки на судебную практику

**Как работает консилиум:**

Система последовательно выполняет несколько этапов:

1. **Сбор мнений экспертов**
   - **GPT 5.2** — глубокий правовой анализ
   - **Gemini 3 Pro** — альтернативная правовая позиция
   - **Claude Opus 4.5** — комплексный анализ (председатель)
   - **Perplexity Sonar Pro** — поиск релевантной судебной практики

2. **Верификация судебных дел**
   - Система извлекает номера дел из ответов экспертов
   - Проверяет каждое дело через официальные реестры (kad.arbitr.ru, consultant.ru)
   - Отмечает подтверждённые и неподтверждённые дела

3. **Экспертная оценка (Peer Review)**
   - Каждый эксперт оценивает ответы других участников
   - Выявляются сильные и слабые стороны каждой позиции

4. **Синтез**
   - Председатель (Claude Opus 4.5) анализирует все мнения
   - Формирует итоговое заключение с учётом всех позиций
   - Указывает на разногласия и обосновывает финальный вывод

**Как использовать:**
1. Выберите режим **«Консилиум»** в переключателе
2. Введите ваш вопрос
3. Нажмите Enter или кнопку отправки
4. Дождитесь завершения всех этапов (отображается прогресс)
5. Изучите итоговый результат

**Просмотр результатов консилиума:**

Результат содержит раскрывающиеся секции:
- **Итоговый ответ** — финальное заключение председателя
- **Мнения экспертов** — ответы каждой модели (можно развернуть)
- **Экспертная оценка** — взаимные оценки экспертов
- **Верифицированные дела** — таблица проверенных судебных дел

## Работа с документами

Система поддерживает загрузку и анализ документов различных форматов.

### Поддерживаемые форматы

| Формат | Расширения | Примечание |
|--------|------------|------------|
| Текстовые документы | .docx, .pdf, .txt | Извлекается текст |
| Изображения | .jpg, .jpeg, .png, .bmp, .gif | Распознаётся текст (OCR) |
| Аудиофайлы | .mp3, .wav, .ogg | Транскрибируется речь |

**Максимальный размер файла:** 25 МБ

### Как загрузить документ

1. Нажмите кнопку загрузки файла (иконка скрепки) рядом с полем ввода
2. Выберите файл на вашем компьютере или перетащите его в область загрузки
3. Дождитесь обработки (извлечение текста, OCR или транскрибация)
4. Под полем ввода появится информация о загруженном файле
5. Введите ваш вопрос по документу
6. Отправьте запрос — текст документа будет автоматически включён в контекст

**Пример использования:**
> Загрузите договор и спросите: «Проанализируй этот договор на предмет рисков для заказчика»

## Сохранение и экспорт ответов

### Сохранение ответа

После получения ответа вы можете сохранить его для последующего использования:

1. Найдите кнопку **«Сохранить»** (иконка закладки) под ответом
2. Нажмите на неё — ответ будет добавлен в вашу библиотеку

### Экспорт в Word (DOCX)

Любой ответ можно скачать в формате документа Word:

1. Найдите кнопку **«Скачать»** (иконка загрузки) под ответом
2. Нажмите на неё
3. Файл .docx автоматически загрузится на ваш компьютер

Документ содержит:
- Ваш вопрос
- Полный ответ системы
- Дату формирования
- Информацию об использованной модели

### Просмотр сохранённых ответов

1. Нажмите кнопку **«Сохранённые»** в верхней панели
2. Откроется список всех ваших сохранённых консультаций
3. Нажмите на любой элемент, чтобы развернуть полный текст

**Доступные действия:**
- **Продолжить чат** — открыть новый диалог с этим ответом в контексте
- **Скачать DOCX** — экспортировать в Word
- **Удалить** — убрать из библиотеки

## Верификация судебной практики

Одна из ключевых функций системы — автоматическая проверка судебных дел.

### Как это работает?

1. Система находит в ответах ссылки на судебные дела (номера дел)
2. Проверяет каждое дело через официальные источники:
   - **kad.arbitr.ru** — Картотека арбитражных дел
   - **consultant.ru** — КонсультантПлюс
3. Отображает результат проверки в таблице

### Статусы верификации

| Статус | Цвет | Значение |
|--------|------|----------|
| Подтверждено | Зелёный | Дело найдено в официальном реестре |
| Не найдено | Красный | Дело не обнаружено в реестрах |
| Ошибка проверки | Жёлтый | Не удалось проверить (технические причины) |

### Таблица верифицированных дел

Для каждого проверенного дела отображается:
- Номер дела
- Статус верификации
- Ссылка на источник (если подтверждено)
- Краткое описание (если доступно)

> **Важно:** Всегда проверяйте статус верификации! Языковые модели могут «галлюцинировать» — придумывать несуществующие дела. Используйте только подтверждённые ссылки.

## Советы по эффективной работе

### Формулировка вопросов

**Хорошо:**
- Чётко сформулируйте правовой вопрос
- Укажите релевантные обстоятельства
- Уточните область права (гражданское, трудовое, налоговое и т.д.)
- Укажите, какой результат вам нужен (анализ рисков, правовая позиция, рекомендации)

**Пример хорошего вопроса:**
> «Компания заключила договор подряда с ИП. Подрядчик систематически нарушает сроки выполнения работ. Какие правовые механизмы расторжения договора предусмотрены ГК РФ? Какие риски для заказчика при одностороннем отказе?»

**Избегайте:**
- Слишком общих вопросов без контекста
- Множественных вопросов в одном сообщении (лучше задавать последовательно)

### Выбор режима

| Ситуация | Рекомендуемый режим |
|----------|---------------------|
| Быстрая справка по законодательству | Запрос → Быстрый |
| Анализ конкретной нормы | Запрос → Думающий |
| Подготовка правовой позиции | Консилиум |
| Анализ судебной практики | Запрос с поиском или Консилиум |
| Сложный спорный вопрос | Консилиум |
| Уточняющие вопросы в диалоге | Запрос → Быстрый |

### Работа с результатами

1. **Критически оценивайте ответы** — AI-модели могут ошибаться
2. **Проверяйте статус верификации судебных дел** — используйте только подтверждённые
3. **Сохраняйте важные ответы** — для последующего использования
4. **Экспортируйте в DOCX** — для включения в документы и отчёты

## Новый чат

Чтобы начать новый диалог с чистого листа:

1. Нажмите кнопку **«Новый чат»** на панели управления

3. Можете задавать новый вопрос

> **Примечание:** История диалога сохраняется на сервере. При необходимости можно вернуться к предыдущим консультациям через раздел «Сохранённые».

## Часто задаваемые вопросы

### Почему консилиум занимает много времени?

Режим «Консилиум» последовательно обращается к нескольким моделям ИИ, проводит верификацию судебных дел и синтезирует результаты. Это занимает от 1 до 3 минут в зависимости от сложности вопроса.

### Можно ли доверять ответам системы?

Система использует передовые языковые модели, но они могут ошибаться. Всегда:
- Проверяйте ссылки на судебную практику (используйте только верифицированные)
- Сверяйте цитируемые нормы с первоисточниками
- Рассматривайте ответы как отправную точку для дальнейшей работы

### Что делать, если система не отвечает?

1. Проверьте подключение к интернету
2. Обновите страницу (F5)
3. Попробуйте отправить запрос повторно
4. Если проблема сохраняется — обратитесь к администратору

### Как получить инвайт-код?

Инвайт-коды создаются администратором системы. Обратитесь к ответственному лицу в вашей организации.

### Сохраняется ли история моих запросов?

Да, история сохраняется на сервере. Вы можете сохранять важные ответы в персональную библиотеку для быстрого доступа.
